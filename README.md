# Anime Manga Trivia

## Features:
This app cap:
* Quiz users on their anime knowledge through a set of questions
* Allow users to choose difficulty level and number of questions 

## Libraries Used
* Butter Knife
* Retrofit2
* Glide
* Google Play Services: AdMob
* Dagger2
* RxJava2
* RxAndroid
* React Libs: RxBinding, RxLifecycle
* Material Spinner
* Apache Commons
